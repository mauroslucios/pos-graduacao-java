# Pós-Graduação em Engenharia de Software com Java e IA

Repositório destinado ao armazenamento de exercícios, projetos, anotações e trabalhos desenvolvidos durante a Pós-Graduação em Engenharia de Software com Java e IA da UNIPDS.

## Objetivos

Aprofundar conhecimentos em Engenharia de Software utilizando tecnologias modernas do ecossistema Java, aplicando boas práticas de desenvolvimento, arquitetura de sistemas distribuídos e Inteligência Artificial integrada a aplicações corporativas.

- Orientação a Objetos

- Multithreading

### Backend Moderno
- Spring Framework

- Segurança de APIs

### Arquitetura de Software
- Microsserviços

### DevOps e Cloud
- Docker

### Inteligência Artificial aplicada ao Java

### Frontend
